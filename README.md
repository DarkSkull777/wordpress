# wordpress
getting site wordpress only from list

```
usage: scanwp.py [-h] [-u URL] [-l LIST] [-o OUTPUT] [-t THREADS] [-sc]

options:
  -h, --help            show this help message and exit
  -u URL, --url URL     Single target URL to scan
  -l LIST, --list LIST  File containing list of URLs to scan
  -o OUTPUT, --output OUTPUT
                        File to save the results
  -t THREADS, --threads THREADS
                        Number of threads to use (default: 60)
  -sc, --status-code    Show status codes```
